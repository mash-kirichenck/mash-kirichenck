### Ключевые проекты

1. **«Развёртывание ВМ в Яндекс Облаке с Terraform и Ansible»**
   * автоматизировала развёртывание инфраструктуры в Yandex Cloud (ВМ, сети, группы безопасности) с применением IaC (Terraform + Ansible);
   * сократила время развёртывания среды с 30 до 5 минут;
   * обеспечила воспроизводимость и масштабируемость решений.

2. **«Запуск кластера PostgreSQL (Active/Standby)»**
   * развернула отказоустойчивый кластер PostgreSQL 14 (Active/Standby) с WAL‑репликацией и автоматизированным failover (Patroni + etcd);
   * добилась доступности сервиса на уровне 99,9 %;
   * время переключения при сбое — менее 30 секунд;
   * настроила мониторинг нагрузки и задержек репликации;
   * протестировала целостность данных при имитации сбоя мастера.

3. **«Cisco Packet Tracer: настройка VLAN и маршрутизации»**
   * настроила сегментацию сети через VLAN 10/20 и ACL: ограничила доступ между подсетями 192.168.10.0/24 и 192.168.20.0/24, обеспечила связь с шлюзами подсетей;
   * реализовала маршрутизацию между площадками (R0/R1);
   * сохранила конфигурацию коммутатора командой `write`.

4. **«Запуск системы мониторинга на базе Zabbix»**
   * настроила систему мониторинга на базе Zabbix (сервер/агенты) с интеграцией MySQL и PHP 8.1;
   * настроила отслеживание доступности и времени отклика yandex.ru (HTTP/ICMP);
   * внедрила алёртинг;
   * протестировала и запустила систему в эксплуатацию.
